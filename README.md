# MIMI-Time

MIMIさんの活動を応援するために制作している非公式のファンサイトです。

## 概要

MIMI Time は、ボカロP「MIMI」(@mimi_3mi)さんの楽曲をきっかけに、 時間帯の雰囲気に合わせた背景とともに、ランダムで選曲される音楽と出会う体験を目的として制作した、 非公式のファンサイトです。

また、このプロジェクトは 多くの方々の協力のもと制作をしています。

## 開発について

このサイトは Python と Flask を使用して制作されています。
